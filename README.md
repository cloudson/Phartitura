![Phartitura](./public/images/logo.png)

### Requirements

* PHP 5.4+ (obviously)
* [Respect tools](http://github.com/respect)
* Redis 
* New Relic extension 


### How run it?  

```bash
git clone https://github.com/cloudson/Phartitura.git phartitura   
cd phartitura
php -S localhost:4000 -t ./public 

```
